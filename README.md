# Библиотека книг

Прописываем глобальную переменную для создания окружения в проекте:
> export PIPENV_VENV_IN_PROJECT=1

Coбрать окружения командами:

> pipenv install --python 3.8
> npm i

В папке с проектом:

> pipenv shell

Запуск сервера:

> python manage.py runserver

Сборка клиента:

> npm run build
